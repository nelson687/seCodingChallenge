Application created using Grails 2.3.10
Instructions
  Run the application with the command "grails run-app" in the application directory.
  Once the application has started, access by browser in localhost:8080
  If you want to receive confirmation emails you can change email address in the two default accounts in  Bootstrap.groovy lines 8 and 9
  To run tests type in the console "grails test-app" and it will run both unit and integration tests.
